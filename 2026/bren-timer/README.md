# bren-timer

Sleep until a specific time, then pop up a zenity warning dialog.

## Install

```bash
./install.sh
# or with a custom destination
PREFIX=/usr/local/bin ./install.sh
```

## Usage

```bash
bren-timer HH:MM [message]
```

```bash
bren-timer 10:00
bren-timer 14:30 "Stand up meeting!"
```
